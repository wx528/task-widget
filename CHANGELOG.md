# 更新日志

## [0.2.0] - 2026-06-27

### 新增

- 每个任务前增加拖动块（≡），支持拖拽调整任务顺序
- 拖放排序后自动保存到 `data/tasks.json`

## [0.1.0] - 2026-06-27

### 新增

- 初始版本发布
- 桌面任务清单小部件，无边框、半透明、圆角、置顶显示
- 支持添加、编辑、删除、完成任务
- 任务数据自动持久化到 `data/tasks.json`
- 精简模式：仅显示待办任务数量
- 系统托盘图标，支持显示/隐藏窗口
- 支持鼠标拖动调整窗口位置
- 提供 PyInstaller 打包配置和开机自启脚本
